# CLAUDE.md - モグる・タベる・イキのこる

## プロジェクト概要
このリポジトリは、ボードゲーム「モグる・タベる・イキのこる」のゲームロジック層を
Pure Kotlin で実装するためのプロジェクトです。

- Android 非依存のロジック層のみを対象にする
- UI 層は含めない
- 後から Android UI を被せる前提で設計する

## 絶対ルール
- 仕様書に記載された機能のみ実装する
- 勝手に新機能や独自仕様を追加しない
- 変更対象は `d:/moguraGame` 配下のみ
- 実装不能・仕様衝突・不明点は省略せず報告する
- `android.*` パッケージは import しない

## ゲームルール要約
### 盤面
- サイズは 6 列 x 5 行
- 有効マスは 26
- 無効マスは `(0,2)`, `(5,2)`, `(0,3)`, `(5,3)`
- 巣は `(0,1)`, `(5,1)`, `(0,4)`, `(5,4)`
- ホットゾーンは `(2,2)`, `(3,2)`, `(2,3)`, `(3,3)`
- 地上は `row=0` の 6 マス

### 穴タイル
- 直線 10 枚
- L字 10 枚
- T字 4 枚
- 十字 2 枚
- 配置時に 90 度単位で回転可能

### エサ
- カブトムシの幼虫: 4枚 / 1点 / 回復1 / 逃走なし
- ミミズ: 3枚 / 2点 / 回復2 / 6面中2面で逃走
- ケラ: 3枚 / 2点 / 回復3 / 6面中3面で逃走
- ムカデ: 2枚 / 3点 / 回復4 / 6面中4面で逃走
- カエル: 1枚 / 4点 / 回復5 / 6面中5面で逃走 / 4人プレイ用

### プレイヤー
- 初期体力は 13
- 地下ではターン終了時に体力 1 減少
- 地上ではターン終了時に体力 2 減少
- 体力 0 で脱落
- 連行中エサは最大 1 枚

### ターンフェーズ
1. DIG: 掘る
2. MOVE: 移動
3. CAPTURE: 捕獲
4. DECIDE: タベる or レンコウ
5. END: ターン終了

### 勝利条件
- 2〜3 人プレイは 4 点先取
- 4 人プレイは 5 点先取
- 全員脱落はドロー

### エサ補充
- ホットゾーン 4 マスに裏向きエサがなくなったら補充する
- ストックが空なら捨て札をシャッフルして再構築する

### 強奪
- 相手の巣に止まり、相手が不在で、相手の巣にエサがあるときに成功する

### 巣防衛
- 自分が巣にいると防衛中とみなす
- 自分が巣へ戻ったとき、侵入者は隣接マスへ追い出される
